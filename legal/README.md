# 法的文書アウトライン(Legal Document Outlines)

このディレクトリには Postkey インスタンスを運営するために必要な法的文書のアウトラインを格納しています。

## 重要な前提

**これらは技術者視点で書かれたアウトラインです**。実際にローンチする前に、**日本の弁護士による法的レビューが必須**です。

## 弁護士レビューが必要な主要論点

以下は技術者だけでは判断できないため、弁護士相談が必須:

1. APPI 2022 改正の詳細解釈(特に「高度な暗号化」例外の適用範囲)
2. EU 居住者除外の法的有効性(GDPR Art. 3(2) のターゲティング基準)
3. ゼロ知識アーキテクチャ下での開示請求への対応方法
4. 個人運営での個人情報取扱事業者としての義務範囲
5. 利用規約の消費者契約法適合性(免責条項の有効範囲)
6. 国際的な令状対応の方針

## 弁護士費用の目安

- スタートアップ向け弁護士サービス(クラウドサイン等): 5〜15 万円程度
- 中堅法律事務所: 30〜80 万円程度
- 個人情報保護委員会の事前相談: 無料(継続的に活用推奨)

## 現実的な進め方

1. **ローンチ前**: アウトラインを文章化 → 低価格弁護士サービスでレビュー
2. **ユーザー増加段階**: 個人情報保護委員会の事前相談を活用
3. **大規模化後**: 専門弁護士と顧問契約
