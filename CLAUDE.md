# Lodester プロジェクト

このプロジェクトは Lodester(OAP プロトコルのリファレンス実装)の実装です。

## 重要: 最初に読むべきファイル

新しいセッションを開始する時は、必ず以下の順で読んでください:

1. HANDOFF.md - 設計フェーズからの引き継ぎ(最重要)
2. DECISIONS.md - 41 の意思決定記録
3. mvp/scope.md - MVP 実装範囲

## プロジェクトの哲学

- ゼロ知識暗号(サーバーは暗号文しか見えない)
- メール型連合(分散による継続性)
- 段階的実装(完璧より完成)
- 個人運営でも持続可能(ベスト・エフォート)

## 言語・技術

- Go 1.23+(私は新規習得中、解説歓迎)
- PostgreSQL 16+
- Linode 東京リージョン
- AGPL v3
